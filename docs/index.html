Carbonite (Project Ascension Edition)

Maintained by Xurkon.

Overview

This is a stabilized version of the classic Carbonite addon, specifically patched to resolve Lua errors and crashes on the Project Ascension 3.3.5a client.

Recent Fixes (v3.34-Ascension)

  • FarmHud Compatibility - Works seamlessly with FarmHud HUD mode.
  • Fixed table index is nil crash when map data is missing (InT1).
  • Fixed nil reference to Win1 window frame (ShU1).
  • Fixed nil map reference in GPP, ToS1, and ALP functions.
  • Fixed nil tooltip options crash in TP2.
  • Fixed number comparison with nil in map update logic (Line 18184).
  • Fixed nil table index crash in SCM1 (Line 23842).
  • Fixed nil local 'inf' crash in ITCZ (Line 6978).
  • Fixed nil GameMenuFrame crash in ShowUIPanel (Line 6871).
  • Disabled "version is old" update notification.

Installation

  1. Download the latest release.
  2. Unzip into Interface/AddOns/Carbonite.
  3. Launch the game and enable the addon.
